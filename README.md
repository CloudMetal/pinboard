# pinboard

Pinterest like app build in node.js

## Install

Install NPM packages:

    $ npm install

Install component:

    $ npm install -g component

Build client-side code:

    $ make

## Run 

Run server:
  
    $ node app.js

View board:

    http://localhost:3000/pinboard

Download bookmarklet:

    http://localhost:3002/pinboard/about
